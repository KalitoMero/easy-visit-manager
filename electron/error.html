Application Error

Die Anwendung konnte nicht korrekt geladen werden. Dies passiert normalerweise, wenn die Build-Dateien fehlen.

Bitte versuchen Sie eines der folgenden Verfahren:

  1. Führen Sie npm run build aus, um einen Produktions-Build zu erstellen
  2. Oder starten Sie den Entwicklungsserver mit npm run dev
  3. Starten Sie dann die Electron-App mit cross-env NODE_ENV=development electron .

Falls das Problem weiterhin besteht, prüfen Sie bitte die Konsole auf detailliertere Fehlermeldungen.
